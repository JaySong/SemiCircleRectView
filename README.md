# SemiCircleRectView
基于TextView实现的一个半圆矩形背景控件

# 特点

 1. 快速实现半圆矩形背景View
 2. 适配所有机型并在任何屏幕尺寸下做到完美的适配
 3. 轻松的实现不同背景颜色
 4. 不需要重复的为不同大小尺寸的背景创建shape.xml
